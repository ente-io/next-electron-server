# Example: Next.js Electron Server

## Download

```bash
$ mkdir next-electron-server-example && cd $_ && curl https://codeload.github.com/HaNdTrix/next-electron-server/tar.gz/master | tar -xz --strip=2 next-electron-server-master/example
```

## Install

```bash
$ yarn
```

## Development

```bash
$ yarn dev
```

## Build

```bash
$ yarn build
```
